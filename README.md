# webProjects-cli

一个快捷创建前端项目的脚手架 项目类型可选 小程序 node服务

# 环境安装

## 1、安装 node 

### v16.2.0 +

## 2、本地运行的话

```
git clone xxx

npm link (在本地将此项目包缓存到本地(npm root -g)，便于在本地调试执行此包)

通过执行 webprojects-cli 创建搭建新项目
```

## 3、全局安装此包 使用

```
npm i webprojects-cli -g

通过执行 webprojects-cli 创建搭建新项目

? 请输入创建包名称 : xxx

? 请输入端口号 (默认 8080） : xxxx
? 请选择要安装的依赖包 koaStatic, koaRouter
```
